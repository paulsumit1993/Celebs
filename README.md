# Installation
- Clone this repo.
- open `Celebs` folder.
- open `terminal` and `cd` to Celebs folder root.
- run `pod install` (install Cocoapods for this command to work [here](https://cocoapods.org/)).
- open `Celebs.xcworkspace` in Xcode 9 Beta.
- <kbd>CTRL</kbd> + <kbd>R</kbd>.
